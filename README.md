# Textbook（日任务笔记）

一个使用 **React + TypeScript + Vite + Tailwind CSS** 构建的轻量「每日计划 / 执行记录」应用。

它的核心目标是：
- 今天专注执行；
- 明天提前规划；
- 每条任务都留下执行状态与复盘笔记。

## 功能特性

- **当日计划视图**：展示当前激活计划与任务清单。
- **任务状态管理**：支持更新任务状态（如未开始 / 进行中 / 已完成）。
- **任务笔记记录**：可为每条任务保存执行备注，便于复盘。
- **明日计划创建**：按规则生成下一天的计划。
- **历史计划回看**：可切换查看过往计划内容。
- **本地持久化**：状态自动保存，刷新页面后不丢失。

## 技术栈

- React 18
- TypeScript 5
- Vite 5
- Tailwind CSS 3

## 快速开始

### 1) 安装依赖

```bash
npm install
```

### 2) 启动开发环境

```bash
npm run dev
```

默认会启动本地开发服务器（通常是 `http://localhost:5173`）。

### 3) 构建生产版本

```bash
npm run build
```

### 4) 本地预览构建产物

```bash
npm run preview
```

## 项目结构（简要）

```text
src/
  components/   # 页面组件（任务列表、计划头部、历史记录等）
  lib/          # 业务逻辑与状态读写
  types/        # TypeScript 类型定义
  App.tsx       # 应用主页面
```

## 使用建议

- 每天结束前创建「明日计划」。
- 执行过程中实时更新任务状态与备注。
- 通过历史记录回看连续几天的完成情况，优化任务颗粒度与优先级。
